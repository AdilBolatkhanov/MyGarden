# myGarden
A gardening app illustrating Android development best practices with Android Jetpack.

## Applied concepts
* MVVM
* own custom DI
* Data Binding
* Navigation
* Lifecycles
* LivaData
* Room
* ViewModel
* WorkManager
* Coroutines
* Glide
* Animations
